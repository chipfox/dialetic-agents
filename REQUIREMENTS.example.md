# Project Requirements: Simple Calculator CLI

## Goal

Create a Python-based Command Line Interface (CLI) calculator.

## Features

1. **Basic Arithmetic**: Support +, -, *, / operations.
2. **History**: Keep a log of the last 10 calculations in a file `history.txt`.
3. **Error Handling**: Gracefully handle division by zero and invalid inputs.
4. **Tests**: Include a test suite using `unittest` or `pytest`.

## Technical Constraints

- Use Python 3.
- No external dependencies for the core logic.
- Code must be in `src/calculator.py`.
- Tests must be in `tests/test_calculator.py`.
